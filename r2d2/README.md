# r2d2

[Using urdf with robot_state_publisher](http://wiki.ros.org/urdf/Tutorials/Using%20urdf%20with%20robot_state_publisher)

Compile:

```sh
catkin_make
```

Run:

```sh
roslaunch r2d2 display.launch
```

Also in another terminal:

```sh
rosrun rviz rviz
```

The result will be:
